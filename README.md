# PDF Merger
Minimal web app to merge multiple PDFs into one document.  
No signup.  
No API.  
Just clean merging.

---

## ✨ Features
- Merge up to **15 PDFs** (max 150MB total)
- Drag & drop support
- **Reorder files** by dragging
- **PDF thumbnails** with page counts
- Custom **file naming**
- Dark/Light **theme toggle**
- Real-time **progress bar**
- **Keyboard shortcuts** (Space to merge, ESC to clear)
- Share your results
- No authentication
- No ads
- No watermarks

---

## 🧠 Tech Stack
- React (Create React App)
- PDF.js (PDF rendering)
- jsPDF (PDF creation)
- Tailwind CSS (Apple-inspired UI)
- Lucide React (icons)
- Vercel (static deploy)

---

## 🖥️ How It Works
1. Upload PDFs (or drag & drop)
2. Everything is processed fully in the browser
3. Reorder files by dragging them
4. Customize the output filename
5. Press Space or click "Merge" button
6. Download your merged PDF instantly

> All PDFs show thumbnails and page counts automatically.

---

## ⌨️ Keyboard Shortcuts
- **Space** → Start merging
- **ESC** → Clear all files

---

## 🔒 Privacy
All processing happens **locally in your browser**.  
No PDFs are uploaded to a server.  
Your files never leave your device.

---

## 📦 Installation
```bash
# Clone the repo

# Install dependencies
cd pdf-merger
npm install

# Run locally
npm start
```

---

## 🌟 Limits
- Maximum **15 files** per merge
- Maximum **150 MB** total size
- High quality output (scale: 2, compression: 95%)
